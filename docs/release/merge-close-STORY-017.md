## Risks / Known Limitations

1. **No update/edit mutation** — Users must delete and re-create to change a secret value. Documented limitation; server API constraint.
2. **Form resets scope to GLOBAL on open** — Minor UX issue, documented by developer.
3. **Hono RPC client uses base Hono type** — Matches existing pattern across all hooks.
